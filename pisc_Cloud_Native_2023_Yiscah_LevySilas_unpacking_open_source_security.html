Unpacking Open Source Security in Public Repos & Registries

Video size:

Abstract

The container ecosystem has exploded in the decade since it’s been introduced and today boasts many public facing resources - open source projects & public registries. This talk we will unveil first of its kind research conducted by scanning the most popular and widely adopted open source projects.
