Beyond Surveys: Real-Time User Satisfaction Prediction with LLMs and Telemetry

Abstract

Stop waiting for surveys! Learn to predict user satisfaction in real time by fusing LLMs with behavioral telemetry. We’ll explore a multi-modal AI framework that identifies “Detractors” with 86% accuracy, enables proactive interventions, and delivers sub-100ms inference with SHAP explainability.
