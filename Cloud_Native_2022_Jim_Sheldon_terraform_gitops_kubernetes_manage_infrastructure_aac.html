Terraform, GitOps and Kubernetes: How to manage your infrastructure and applications as code

Video size:

Abstract

Everyone agrees that infrastructure as code is vital to managing cloud resources, but what are the best ways to arrange and manage that code? How does GitOps fit into the picture? Jim will share different approaches that he has seen development teams adopt, to be successful in managing not only their cloud infrastructure, but their Kubernetes resources as well.

Jim will cover git repository structure and layout, as well as when it makes sense to split configuration into multiple repositories. Once the structure is created, Jim will go over how to best test and apply the configuration changes from development to production.
