Abstract

Explore Kubernetes security on multiple levels: Start with the Infrastructure, using firewalls and patches. Move to the Cluster with RBAC and Calico. Containers with images scan for vulnerabilities. Secure applications with Istio, and ensure code compliance through reviews and Open Policy Agent.
