Agentic AI for Autonomous Micro-Frontend User Interfaces and Microservices Evolution in Cloud Platforms

Video size:

Abstract

Cloud-native org. rely on microservices for backend modularity and micro-frontends for scalable user interface delivery. Yet, real-world systems still struggle to evolve. I will talk on a unified Agentic AI framework that autonomously coordinates the co-evolution of micro-frontend UIs using ReactJS.
