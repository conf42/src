Kubernetes Deployment from Platform to API

Video size:

Abstract

Organizations that adopt Kubernetes aim to improve productivity and the developer experience when deploying applications. We will explore various perspectives on application deployment in Kubernetes and highlight the open-source tooling and AWS solutions that enable these advancements.
