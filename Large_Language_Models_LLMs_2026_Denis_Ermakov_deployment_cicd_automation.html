Continuous Agentic Deployment: An Open-Source Framework for AI-Powered CI/CD Pipelines

Video size:

Abstract

LLM agents can generate code, but deploying them reliably in real development workflows requires more than prompting a model. Teams need structured pipelines, observability, retry mechanisms, and clear points where humans stay in control. In this talk, Denis introduces Continuous Agentic Deployment (CAD)  an open-source framework for integrating AI agents directly into CI/CD pipelines. The framework combines GitHub Actions, prompt libraries, evaluation steps, and cost dashboards to orchestrate specialized agents responsible for issue analysis, code generation, testing, and deployment. Well walk through a real architecture for running agent-powered pipelines, including patterns for retries, validation steps, human review gates, and metrics for tracking cost and throughput. The session also covers lessons learned from early experiments, including where automation fails and how to design safe fallbacks. Attendees will leave with a practical blueprint for deploying AI agents in production pipelines while maintaining reliability, observability, and engineering discipline.
