Doing more with Cloud API's

Video size:

Abstract

All the major Cloud vendors provide customers with a rich api based capability to manage your environment. In many cases these apis provide more functionality than is available through the Web console, but less easy to use. By extracting api output and loading into a database this allows you to perform complex analysis on that data, and build applications to enhance your Cloud experience
