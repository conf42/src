How to hack a cloud production environment with external Terraform manipulation

Video size:

Abstract

The power of Terraform providers & modules are undeniable, but so are the potential security risks if not properly managed. This session covers how widely-used Terraform open source components can be exploited to gain full control of a cloud production environment and best practices for mitigation.
