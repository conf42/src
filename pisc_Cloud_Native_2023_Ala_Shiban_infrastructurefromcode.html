The State of Infrastructure-From-Code 2023

Video size:

Abstract

This talk will explore Infrastructure-from-Code (IfC), a method of creating and managing cloud infrastructure from application code, and its effect on developer-operator productivity in 2023. We’ll look at the tools, approaches and how it might reshape the devops movement altogether.

Summary

  • infrastructurefromcode from code enables teams to write universal backend code that runs anywhere. It refers to tools that automatically infer the needed infrastructure components from application code. There are four dominant approaches to realizing this aspiration.
