Building a Production Computer Vision Verification System for Warehouse Receiving

Video size:

Abstract

I will present a case study of designing and deploying a computer vision verification system for warehouse receiving in an industrial supply chain. The session focuses not only on detecting visually similar spare parts, but also on how the solution was embedded into an existing operational workflow under real production constraints. It covers the end-to-end design of an AI-assisted receiving process, including image-based verification, the use of catalog and technical reference data, and integration with warehouse operations. Special attention is given to confidence thresholds, exception handling, human-in-the-loop validation, service reliability, and the trade-offs between accuracy and throughput in a time-sensitive environment. Attendees will also see how the system was operated in practice, how user interaction was designed for warehouse personnel, and how the deployed solution improved receiving quality and operational consistency while reducing spare-parts mismatches by 99%.
