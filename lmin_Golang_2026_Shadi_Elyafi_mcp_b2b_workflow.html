Building an MCP Server in Go for a Real B2B Workflow

Video size:

Abstract

In this talk, I’ll show how I built an MCP server in Go for a real B2B workflow used inside a platform serving more than 1 million company profiles.

I’ll walk through how the server exposes a set of tools for Claude, connects to a GraphQL backend through AWS AppSync, and handles authentication safely in a production-oriented setup.

The session covers practical implementation details such as tool design, read/write boundaries, mutation safety, and how to structure integrations so LLM-driven workflows remain reliable instead of becoming fragile demos.

I’ll also explain how I approached testing with TDD, including mock GraphQL clients and API-level validation, so the MCP layer can evolve without breaking core behavior.

Attendees will receive a concrete path for building Go-based MCP services that connect real business systems, along with lessons on GraphQL integration, AWS authentication, and production-friendly testing patterns.
