Cloud Native Posture Management for All

Video size:

Abstract

This talk introduces two symbiotic open source projects, cnquery and cnspec, that bring a cloud native approach to security posture management across all parts of your infrastructure.

Summary

  • Mondoo's Ben Rockwood talks about cloud native posture management for all. He introduces two new open source projects that he thinks will have a profound impact on the industry. Security just means protecting what matters. Compliance is really about doing what we say that we do.
  • This is one of the critical things about security is being able to have an awareness of what's in our environment. The heart of Kubernetes is this idea of a reconciliation loop. The best way to get extensible is to make sure that we're open source because we really want to get towards assurance.
  • Security posture issues come down to two sort of things. One is running outdated software, software that's just vulnerable. The more egregious one is misconfiguration. There needs to be a more continuous way of being able to audit our infrastructure.
  • Cnquery is a cloud native query tool. It's a graphql interface to all the things. Can filter by name, uid and Shell. Can pull back all the data that's available. Can test all kinds of different aspects of an operating system.
  • CNquery and CNSPEC abstracted all those APIs, all those clouds, your operating system and all kinds of other things in such a way that we can interface with it in the same way no matter what it is that we're interfacing with. This is what we think of as extensible security posture management, XSPM.
