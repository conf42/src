Building Machine Learning-powered Applications with JavaScript

Video size:

Abstract

When dealing with machine learning (ML) requirements, most teams generally start using Python or R due to the number of references available using those languages. However, there are cases where companies do not have the bandwidth and budget to learn a new language on top of JavaScript especially if JavaScript and Node.js are being used in most deployed systems. That said, ML practitioners should still be able to build and deploy ML-powered applications using JavaScript as long as they know which tools, services, and packages are available for these types of requirements.

In this talk, we will discuss how to use different techniques to build machine learning powered apps ranging from fully abstracted to fully customized solutions with JavaScript
