AI-Driven Engineering Quality: How We Use Machine Learning to Predict Delivery Time and Improve Code Reliability

Video size:

Abstract

Shipping breaks when planning is guesswork. I’ll show a production ML system that scores pull requests for risk, estimates delivery time before coding starts, and flags bottlenecks, using repo metadata, AST features, test/CI signals, and LLM-based code smells, fully integrated into CI.
