How the magic of open source helps our project

Video size:

Abstract

How many external libraries do you use in your app? Have you ever wondered how they are there for you for free? Have you ever realized you can dive into the code and see what they do? And that you can patch them to make them do exactly what you want? Learn more about this in this talk!

Summary
