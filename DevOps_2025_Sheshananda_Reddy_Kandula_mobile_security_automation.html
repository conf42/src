Mobile Application Security Assessments Automation with MobSF and Jenkins pipeline

Video size:

Abstract

With mobile app usage and attacks on the rise, automating Mobile AppSec is essential. MobSF, a powerful security platform for Android and iOS, integrates seamlessly with Jenkins pipelines to automate security assessments. This ensures consistent, efficient testing, strengthen app security at scale.
