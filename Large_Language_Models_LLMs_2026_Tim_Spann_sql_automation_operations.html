Abstract

With Cortex Code and the power of DBT, SQL, Python and autonomous agents we can easily manage and build advanced automation, continuous integration, continuous deployment and version control. By using Snowflake authored skills, Cortex Code CLI allows for natural language to manage schema creation and change, with full testing and automated deployments.
