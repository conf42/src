Securing Git Repositories: Preventing Accidental Secret Commits with Gitleaks

Video size:

Abstract

Accidentally committing secrets to a Git repository can compromise sensitive data and lead to serious security risks. This talk will focus on addressing this common issue by using Gitleaks, an open-source tool for detecting secrets in code, and demonstrate how developers can seamlessly integrate it into their workflows. Instead of scrambling to fix the problem after secrets have already been exposed, this session will show you how to catch potential leaks before they happen. Through a live demo, we will explore how Harness Open Source provides a streamlined local DevOps platform to enhance developer productivity while maintaining strong security measures. Attendees will learn how to safeguard their repositories, avoid accidental exposures, and use open-source tools to improve overall development practices.
