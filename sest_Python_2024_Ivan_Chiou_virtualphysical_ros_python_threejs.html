Implementing a Virtual-Physical Environment Manipulation System based on ROS using Python and Three.js

Abstract

By leveraging Python+rosbridge and VueJS+Three.js, it becomes straightforward to create a ROS-based Digital Twins system. Through the integration of WebRTC and IP cameras, this system allows the operation and training of physical robotic arms on a cloud-based webpage, with Dockerization deployment.
